# SentinelAI

### AI-Powered Cyber Defense Copilot for Critical National Infrastructure

SentinelAI is an AI-powered cybersecurity dashboard that assists Security Operations Center (SOC) analysts by simulating real-time security events, assessing cyber risks, mapping attacks to the MITRE ATT&CK framework, and generating AI-driven incident reports.

Built for the **ET AI Hackathon 2026**.

---

## Problem Statement

Critical infrastructure such as power grids, government agencies, and water treatment facilities generate thousands of security events every day. Manual log analysis is time-consuming, increasing the risk of delayed threat detection and incident response.

SentinelAI leverages Large Language Models (LLMs) to automate log analysis and provide rapid, actionable insights for security analysts.

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/ethanrebeiro/SentinelAI.git
```

2. Open `script.js`

3. Replace:

```javascript
const API_KEY = "YOUR_GROQ_API_KEY";
```

with your own Groq API key.

4. Open `index.html` in your browser.
